# Quant Trading 📈

量化交易策略与工具集

## 目录结构

```
quant-trading/
├── strategies/          # 交易策略
├── data/               # 数据处理
├── backtest/           # 回测框架
├── utils/              # 工具函数
└── config.py           # 配置文件
```

## 策略列表

- [ ] 均线交叉策略 (MA Crossover)
- [ ] 动量策略 (Momentum)
- [ ] 均值回归 (Mean Reversion)
- [ ] 配对交易 (Pairs Trading)

## 使用

```bash
pip install -r requirements.txt
```
